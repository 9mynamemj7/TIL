# fix
1. 왜 아래로 내려가 있는거 같지 (index.html ::33)
    | baseline 그리고 좌측, 중앙, 우측, 상단, 중간, 하단 정렬 개념을 알 필요가 있다. text baseline은 인터넷에 검색해 보자!
    텍스트 그리고 div와 같은 html element 들은 모두 부모 요소 안에서 배치하는 방법을 결정할 수 있는 css 속성이 있다.

    * align, flex, gird 등 
2. px 단위 말고, em, rem, vh, vw의 단위를 알아보자
    | 대 모바일 시대, px 고정값으로 css를 박아버리면 다양한 모바일 디바이스에 대응하기 힘들어 진다. 흔히 반응형이라고 부르는 것에 대해 알아둬야 한다.
